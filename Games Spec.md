# Sight Words — Games Spec (v1.5 "Play" overhaul)

Companion to `Sight Words - PRD.md`. This spec is the binding contract for the game build. Inspiration source: Lucas & Friends screenshots review (briefs in session; originals in `_inbox/feautre inspiration screenshots/`). Justin's locked decisions: ALL 5 games this pass (Word Hunt / Say & Match / Memory extra-polished; Missing Letter / Spelling Builder lighter review); fox & owl splash stays; guided session ≈ 7 min (cards + 1 game round); deploy to dad's iPad only; NO host characters anywhere — instructions are Rachel's voice + glow/pulse affordances, never an on-screen character.

## 1. Shared conventions (every game, non-negotiable)

- **InstructionSpeaker**: fixed top-left darkPlate speaker button on every game screen. Tap → replays the current instruction line (and re-highlights whatever the instruction refers to, where applicable). Animates (subtle pulse) while any Rachel line plays.
- **Tap-to-hear everywhere**: tapping any rendered word says the word; tapping any letter tile says the letter name (Missing Letter tier 2+ uses letter sounds — see §4). Never gated.
- **SuccessMoment** (shared component): the found/completed word zooms large center-screen in its highlight color, Rachel says it, confetti bursts (reuse/extend ConfettiBurst), ~1.4s, then settles. Same beat in every game.
- **Wrong-answer treatment** (shared): target shakes once, soft "boing" SFX, element returns/resets. NO scolding voice, no red X, no lost progress. (Their "no no no" line is explicitly rejected — conflicts with our no-punishment principle.)
- **RoundCelebration** (shared): round end → full-screen confetti + rotating praise clip (4 variants, never same twice in a row) + pulsing Next key (ChunkyKey accent). Set end → return to Games shelf or continue guided session.
- **Exit gate**: X top-right requires press-and-hold ~0.6s (radial fill indicator). Mid-game exit persists nothing beyond already-recorded exposures.
- **Voice moments**: any game step marked 🎤 uses VoiceCheckService with the same rules as solo cards: confident match auto-advances, near-miss → confirm bar, silence → after 6s gentle prompt + the step becomes skippable via a "Show me" fallback button. Voice steps REQUIRE profile.voiceCheckOn; when off/unavailable the 🎤 step is skipped entirely (games stay fully playable) — never block play on the mic.
- **Layout**: iPad landscape primary (kids), iPhone compact functional via existing isCompact patterns. Board on an orange-bordered rounded white card (our Theme colors, cornerRadius 22+) over the new warm landscape backdrop (WP-G2 art; WarmBackdrop as fallback via Art.exists).

## 2. Architecture (contracts for parallel workers)

New files only — existing Engine files remain frozen; game logic that is pure goes in NEW Engine files with smoke tests.

- `Sources/Engine/GameTiers.swift` (pure): `enum GameTier: Int (t1/t2/t3)`; `struct TierLadder` — staircase state machine: promote after 5 clean rounds of last 7 (clean = 0 timeout-hints and ≤1 wrong attempt), demote after 2 consecutive rough rounds (rough = ≥3 wrong attempts or ≥2 timeout-hints), backstop-promote after 15 rounds at same tier with ≥90% clean, NEVER promote while `reviewBacklogGrowing` flag true (fed by LearningService: needsReview count rising vs 7 days ago). Tier changes only between rounds.
- `Sources/Engine/GameWordPicker.swift` (pure): given pool snapshots + count + constraints (maxLength for grids, distractor needs), returns round words weighted: learning/developing 60%, fluent/mastered 30%, new 10% (games INTRODUCE nothing — new words only appear if already introduced in cards; if pool lacks introduced words, fill with familiar). Also `confusables(for:)`: homophone-group members + edit-distance-≤2 same-length neighbors from the active pool (for tier-2+ distractors).
- `Sources/App/Games/GameKit/` (WP-G1): `GameCatalog` (all 5 games registered with id, title, icon, tier storage key — game workers DO NOT edit this file; slots pre-stubbed), `GameScaffold` (backdrop, board card, InstructionSpeaker, exit gate, round/celebration chrome), `SuccessMoment`, `WrongShake`, `GameSessionRecorder` (records light exposures: `timesSeen`+1 and lastSeenAt only — never state transitions/fluent days; per-game tier + round history on Profile via `gameTierData: Data` additive field).
- `Sources/App/Games/<GameName>/` — one folder per game, game workers touch ONLY their folder.
- Guided weave (WP-G8): session = existing card session (~8 cards) then 1 game round (game rotates by day; uses that session's due words via GameWordPicker) then SessionComplete. Coordinator gains a `.gameRound` phase wrapper hosting the game view with a "guided" flag (no shelf return; feeds into complete screen).

## 3. The five games

### 3.1 Word Hunt (word search) — POLISH TIER
Loop: Rachel "Find the words!" → 3-5 list words shown right panel (tap any = hear it) → child swipe-selects contiguous letters; commit on finger-lift. Correct: SuccessMoment + 🎤 "Now you say it!" (voice confirm → word checks off with a flourish; voice off → checks off after SuccessMoment). Wrong path: selection ribbon fades, no penalty. Idle >hint-delay: an unfound word's letters pulse 3× in place. Solved trails persist in pastel colors. All words found → RoundCelebration.
Tiers: T1 5×5, 3 words, across/down, hint 8s. T2 6×6, 4 words, +diagonals, hint 12s. T3 7×7, 5 words, confusable decoy letters seeded near targets, hint on demand only (tap the word in list twice). NO reversed words ever. Grid gen must guarantee solvability + no accidental profanity (word list is our safe Dolch pool; decoy fill letters random weighted to target letters).
Juice: elastic neighbor-letters shift away from finger path; letters lift under touch.

### 3.2 Say & Match (listen & match) — POLISH TIER
Loop round A (hear→find): Rachel "Which word did you hear? … <word>" → 3 word tiles rise gently (no balloons; simple floating tiles with soft bob) → tap answer. Correct: wrong tiles drift off-screen, correct tile centers → SuccessMoment. Wrong: shake + Rachel replays word (via InstructionSpeaker auto-fire). Round B (see→say) 🎤: single word tile centers, "Your turn — read it!" → voice confirm → SuccessMoment. Rounds alternate A/B through the word set (B skipped if voice off).
Tiers: T1 3 choices, dissimilar distractors. T2 3 choices, confusable distractors. T3 4 choices + B rounds required (voice on) / T3 with voice off: 4 choices + faster tile bob.

### 3.3 Memory Match — POLISH TIER
Loop: face-down cards grid, "Match the cards!" → flip two; every flip speaks its word. Match: cards stay up, confetti at both, 🎤 optional "Say it to bank it!" (voice confirm adds a star on the pair; pure bonus, no penalty). No match: soft shake, flip back. Board clear: bloop-out, next round bloops in, RoundCelebration at set end.
Tiers: T1 3 pairs (2×3). T2 4 pairs (2×4). T3 5-6 pairs AND pairs become word↔speaker-icon cards (tap speaker card = hears word; match print to sound).

### 3.4 Missing Letter — LIGHTER REVIEW
Loop: worksheet card, 4-6 words each missing letter(s) shown as glowing blank; tray of letter tiles below (with decoys). Tap tile = letter name (T1) / letter SOUND (T2+, see §4 fallback). Drag to blank: correct → letter locks, word reflows seamless, word spoken, mini-confetti. Wrong → tile returns + blank wiggles ("no" head-shake motion, no voice). Board complete → RoundCelebration.
Tiers: T1 one blank/word, +1 decoy. T2 confusable-letter decoys (b/d, p/q, m/n). T3 two blanks on some words, bigger tray.

### 3.5 Spelling Builder — LIGHTER REVIEW
Loop: target word slots + scrambled tiles tray. 🎤 T1 opener: "Say the word, then build it!" (voice confirm unlocks tiles with a sparkle; voice off → tiles just unlock). Drag tiles to slots; correct snaps+thunk; complete → segments fuse into one solid green word pill (their best moment — keep) + word spoken + confetti.
Tiers: T1 word's own letters only. T2 +2 distractor letters. T3 word shows 2s → hides → build from memory ("look, say, cover, check"); peek button available (counts as a hint).
